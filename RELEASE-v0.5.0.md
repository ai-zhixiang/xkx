## 📌 版本概述

v0.5.0「侠客行」是享客虾首个具备 **Agent 引擎** 的测试版。  
付费用户获得独立的 Hermes Agent 子进程，拥有搜索、文件记忆、PDF 文档生成等能力。  
免费用户走 DeepSeek 直调 + 付费引导。

---

## 🆕 核心功能

### 🤖 Agent 多租户引擎
- 每位付费用户分配独立 Agent 子进程（Worker v6）
- 子进程空闲 15 分钟自动回收，最大并发 20 进程
- 异常崩溃自动重建

### 🧠 两级记忆系统
- **近期对话**：history 参数注入，base 5 条 / standard 15 条
- **长期记忆**：`save_memory` 工具写入 `user_facts.json`，重启恢复
- 按套餐分级：basic 5条记忆 / standard 20条记忆 + 文档生成

### 🔍 工具能力
- DuckDuckGo 实时搜索
- 文件记忆持久化
- `save_document` → PDF 文档生成（wkhtmltopdf）→ 静态下载链接
- basic 套餐拒绝文档生成，引导升级

## 🔧 技术架构

```
用户消息 → 微信回调 → session_manager
    ├── 免费用户 → DeepSeek API 直调 + 引导付费
    └── 付费用户 → Agent Worker（子进程）
        ├── 工具搜索（DuckDuckGo）
        ├── 文件记忆（save_memory）
        └── 文档生成（save_document → wkhtmltopdf → PDF）
```
