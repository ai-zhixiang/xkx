## 一、概述

### 核心理念

**不让用户填表，让用户玩卡。**

种子卡是嗨卡的终极形态——有生命、有成长、有故事。

但v1.3有一个更清晰的使命：**「玩别人的免费，创造自己的付费。」**

所有人可以在公益池子里领养、养别人的种子。但想自己创造一颗种子进入池子——需要享客虾会员（¥9.9/月）。

### 免费 vs 付费一览

| 行为 | 免费用户 | 享客虾会员（¥9.9/月） |
|------|---------|-------------------|
| 浏览种子池 | ✅ | ✅ |
| 领养别人的种子 | ✅ | ✅ |
| 养5天 → 开花 → 配捐 | ✅ | ✅ |
| 每日分享打版（官方种子） | ✅ 20颗/天 | ✅ 不占名额 |
| **自己创造种子投池** | ❌ | ✅ |
| **领养会员专属种子** | ❌ | ✅ |
| 个性化定制享客虾对话指引 | ❌ 受限版 | ✅ 完整版 |

### 为什么种子要付费

- v1.3核心目标是「攻克付费转化」——种子系统如果是全免费，转化从何而来？
- 每日分享打版已有20个免费名额可玩，普通用户足够体验种子循环
- 不想付费的用户就在公益池子里玩、养花、比赛——社交价值不差
- 想把自己想法变成种子的人→自然需要享客虾→自然走到¥9.9

### 种子 vs 旧嗨卡对比

| 对比 | 旧嗨卡（v1.2） | 种子卡（v1.3） |
|------|---------------|---------------|
| 生命周期 | 制作→发送→结束 | 种→漂→养→开花→转赠→回种 |
| 参与感 | 制卡人单方面输出 | 种植人+领养人双向互动 |
| 知识传递 | 无 | 5天路径解锁，每节新鲜内容 |
| 公益嵌入 | 无 | 每颗种子自动配捐 |
| 付费转化 | 菜单栏推销 | 想创造种子→享客虾→会员转化 |

## 二、两种种子类型

### 2.1 官方种子（每日分享打版）

**定位：** 冷启动示范 · 让用户看懂种子怎么玩，**且完全免费**

**来源：** 每日分享经典诗文池（30篇起，每周一+20篇）

**为什么免费：** 官方种子是公益池的底座，让所有用户（包括非会员）都能体验种子系统的全流程——领养→养→开花→配捐。官方种子不参与「创造」行为，所以免费合理。

**路径体系：** 每首诗预配 3 条路径，每条 5 天

| 路径 | 侧重 |
|------|------|
| 路径A | 创作故事线（作者生平+创作背景+历史现场） |
| 路径B | 文本赏析线（结构分析+历代点评+翻译对比） |
| 路径C | 文化影响线（书法/影视/现代改编/传播轶事） |

**分配：** 领养时随机分配一条路径，用户无选择权

**名额：** 每天 20 颗，免费用户专属。会员可领但不占名额。

**产出：** 开花后得到经典诗文嗨卡（免费基础版）

### 2.2 自定义种子（享客虾驱动 · 需会员）

**定位：** 真正好玩的部分 · 用户创意释放 · **付费转化的核心**

**为什么要会员：** 自定义种子是用户用脑海里的想法，让AI帮忙变成一颗可以漂流、被领养、被养的种子。这是「创造」行为，是付费壁垒。

**享客虾 × 嗨卡 交互构想：**

享客虾不只是"助手"，它是种子卡系统的驱动力核。以下三个场景说明享客虾如何串联种子和嗨卡：

**场景A — 从享客虾对话直接种一颗种子（会员流程）**

```
用户：「我刚看到一句诗特别有感觉」
享客虾→ 检测到非会员 → 温馨提示「自己创造种子需要享客虾会员哦」
→ 用户表示想开 → 引导支付¥9.9开通享客虾
→ 会员后 → 调用AI编排5天路径 → 弹出预览卡
→ 用户确认 → 种子投池
```

交互要点：
- 享客虾在对话中主动识别用户的"灵感时刻"，即时生成种子预览
- 预览卡可交互：左右滑动看D1→D5内容摘要，不满意可让享客虾重新编排
- 确认即投池，零页面跳转
- **非会员在预览步骤被温和引导到付费**

**场景B — 从每日分享页衍生自定义种子（可引导会员）**

```
用户读到一首每日分享的诗 → 觉得意犹未尽
→ 底部「🌱 找享客虾深挖这首诗」
→ 非会员 → 享客虾提示「自己深挖做种子需会员，要先开通吗？」
→ 会员 → 直接带上下文进入享客虾对话 → 设计专属路径 → 投池
```

交互要点：
- 每日分享页让免费用户尝到种子好玩的甜头
- 「找享客虾深挖」是转化触点——用户想更进一步时，温和告知付费门槛
- 保留了一条用户自然走上付费的路径，而非硬墙

**场景C — 种子开花后直接做成嗨卡**

```
种子开花 → 开花页面底部：
  🌸 开花！
  🎴 做成嗨卡 → 进入hai.html（自动带入开花的诗/图/路径记录）
  🌱 种回池子继续漂
  📁 收进个人花园
```

交互要点：
- 开花🎴嗨卡是种子系统的出口——种子养完了，成果是一张可以发出去的嗨卡
- 💡 做嗨卡是免费的——会员是创造种子的时候付，不是消费的时候付
- 享客虾在"做成嗨卡"这一步可协助配歌、选版式（在对话里问一句"想要什么风格？"）
- 没有"强制做成嗨卡"，用户可选种回池子继续社交循环

**享客虾对话驱动的核心原则：**
1. **用户说想法，享客虾做方案** — 不打字填表，没有多步表单
2. **预览确认即可** — AI生成内容后弹预览卡，用户点头就投池
3. **上下文自动带入** — 从每日分享/音乐厅/花园进来找享客虾，相关信息自动携带，用户不必重复说
4. **自然引导付费** — 检测到非会员尝试创造时，享客虾温馨引导，不是硬弹窗

### 种子与嗨卡的关系

```
种子 = 有生命周期的社交载体（种→漂→养→开花→转赠→回种）
嗨卡 = 种子开花后的静态成品（可发朋友圈、可转赠）
享客虾 = 贯穿全程的AI驱动力 + 付费转化引擎

免费用户：领养别人的种子 → 养5天 → 开花 → 嗨卡
会员用户：创造自己的种子 → 投池 → 被领养 → 开花 → 嗨卡
```

### 自定义种子可设定的解锁内容

| 分类 | 示例 | 需要会员才能用？ |
|------|------|----------------|
| 文字/故事 | 日记、信、诗 | ❌ 会员可自由设 |
| 图片 | 照片、插画 | ❌ 会员可自由设 |
| 语音 | 自己说的话 | ❌ 会员可自由设 |
| 音乐 | AI写歌、选曲库 | ✅ ¥9.9/月会员 |
| 问答/谜题 | 猜歌词、猜作者 | ❌ 会员可自由设 |
| 混合 | 以上任意组合 | 按需 |

**关键规则：**
- 创造种子本身就是会员特权——不管种子内容是什么
- 自定义种子中的音乐/高级内容额外需要会员才能使用（如果用户已经是会员则直接可用）
- 享客虾在对话中会设定路径，用户只需说想法

## 三、种子生命周期

### 3.1 种（入池）

| 种子类型 | 谁可以种 | 条件 |
|---------|---------|------|
| 官方种子 | 所有用户（每日分享页「种下去」按钮） | 免费，每天限1颗（会员2颗） |
| 自定义种子 | **仅享客虾会员** | 需会员订阅¥9.9/月 |

- 官方种子：每日分享页一键「种下去」
- 自定义种子：享客虾对话完成 → 确认投池（**非会员在此步被引导付费**）
- 每人每天可种 1 颗官方种子（会员 2 颗）
- 自定义种子不限量（但享客虾对话自然节奏会约束用户不会一天种上百颗）
- 入池时带路径设定（官方种子随机3路取1，自定义种子由享客虾协助设定）

### 3.2 漂（入公共池）

- 所有种子进入同一公共池
- 池子按「待领养」展示
- **免费用户只能看到并领养免费种子**（官方种子 + 自定义种子中创作者设为免费的）
- **会员专属种子带🦞标识**，领养也需要会员身份

### 3.3 捞（领养）

| 领养类型 | 谁可以领 | 条件 |
|---------|---------|------|
| 领养免费种子 | 所有用户 | 0成本 |
| 领养会员专属种子 | **仅享客虾会员** | 需会员订阅 |

- 每人同时最多领养 3 颗（不区分免费/付费用户）
- 每天可操作 1 颗种子的解锁
- 领养是社交行为，不收费——即使会员专属种子，领养本身也不额外收费

### 3.4 养（5天解锁）

| 天数 | 用户操作（30秒） | 系统推送 |
|------|----------------|---------|
| D1 | 听/看当天内容 | 路径第一节 |
| D2 | 写一句感受（可选） | 路径第二节 |
| D3 | 转发给一个朋友（可选） | 路径第三节 |
| D4 | 选最喜欢的部分 | 路径第四节 |
| D5 | 点「开花」按钮（仪式） | 🌸 开花仪式 |

**规则：**
- 养种子完全免费——不区分免费用户和会员
- 每天不强制互动，不看就不开花，但种子不消失
- 中断超过 3 天 → 种子退回公共池
- 退回的种子保留已有进度，新领养人接续

### 3.5 开花（终点）

```
🌸 恭喜开花！

🎖️ 公益轨迹
  种子：《XXX》
  种植人：张三
  领养人：李四
  天数：5天
  配捐：¥0.10
  累计：¥Y.YY
  标签：[公益小达人]

📋 成长记录
  D1 [冷知识] ✓
  D2 [文化故事] ✓
  D3 [历史回响] ✓
  D4 [跨界呈现] ✓
  D5 [开花配捐] ✓

💝 这颗花可以：
  [🎴 做成嗨卡发出去]  ✅ 免费
  [🌱 种回池子继续漂]  ✅ 免费
  [📁 收进个人花园]    ✅ 免费
```

开花仪式完全免费——**付费只在「创造种子」这一步**。

### 3.6 转赠与回种

- 开花的花卡可以像普通嗨卡一样转赠给朋友（免费）
- 也可以「种回池子」——以种子形态再漂一次（带已有配捐记录）
- 转赠/回种不限制次数
- 回种的花卡，被新的人领养后，产生新的配捐

## 四、公益嵌入

### 4.1 公益金配捐机制

每颗种子从种下到开花持续产生公益金：

| 节点 | 公益金 | 资金来源 |
|------|--------|---------|
| 种下 | ¥0.01 | 平台补贴 |
| 被领养 | ¥0.02 | 享客虾公益基金（平台从会员收入划拨） |
| 每日解锁 | ¥0.01/天 | 平台补贴 |
| 开花 | ¥0.05 | 平台补贴 |

**单颗种子合计：¥0.10**

合作方投放公益种子到池子，开花 = 配捐兑现。

### 4.2 关于「音点」——砍掉，不拿回来

> **❌ 音点系统已砍（v1.2已清理），v1.3不拿回来。**

原文档出现「领养人贡献1音点」是笔误——音点系统在v1.2已随免费化彻底移除。v1.3的公益资金全部走以下渠道：

| 资金来源 | 说明 |
|---------|------|
| 平台补贴 | 运营成本，从整体收入中划拨 |
| 享客虾公益基金 | 会员费（¥9.9/月）按固定比例（建议5%-10%）注入公益池 |
| 用户自愿捐款 | 独立募捐入口，见4.3节 |

**为什么音点不拿回来：**
- v1.2砍音点的原因成立——让用户理解「官方有固定价值体系」增加认知负担
- v1.3种子系统的付费只有一个标准：享客虾会员。不再有多元积分体系
- 领养行为本身是情感驱动（好奇/共鸣/善意），加一道积分门槛反而降低参与率

### 4.3 独立募捐入口（新增）

在每日分享页、花园、开花仪式页增加独立捐款入口：

```
                     🌱 公益种子
                          │
    ┌─────────────────────┼─────────────────────┐
    │                     │                     │
 官方种子            自定义种子              公益种子
 看→养→开花（免费）  创造→投池（需会员）   合作方投放·开花即配捐
                                      （用户也可直接捐款）

                        ↓
               独立捐款入口（全局可见）
              ┌──────────────────────┐
              │  💰 为每颗种子捐款    │
              │  ¥1 | ¥5 | ¥10 | 自定义 │
              │  ┌──────────────────┐ │
              │  │  ❤️ 立即捐款     │ │
              │  └──────────────────┘ │
              │  已有N位爱心人士捐款  │
              └──────────────────────┘
```

**设计要点：**
- 入口位置：每日分享页底部、个人花园右上角、开花仪式页底部
- 每次捐款→自动绑定到用户最近领养/种植的种子
- 捐款记录：种子的成长轨迹上会显示「XX为此种子捐款¥X」
- 不强制、不弹窗、不阻断——静默入口，情感驱动
- 款项去向：享客虾公益基金（需明确公告，建议每季度公示）

**合作方投放：** 公益组织可在池子里投放带配捐承诺的公益种子，用户领养→5天开花→触发合作方配捐。合作方不再限定为诗文，可以是公益主题内容（环保/教育/健康等）。

### 4.4 公益金可视化

每个用户的主花园展示：
- 累计公益贡献（¥）
- 被自己花朵滋养的种子数
- 参与捐款次数的爱心徽章

## 六、付费转化漏斗

### 整体结构

```
  🆓 免费用户层
├── 每日分享打版 → 领养官方种子 → 养5天 → 开花 → 嗨卡
├── 浏览种子池 → 看别人的创意
├── 领养免费自定义种子
└── 参与公益种子配捐
        │
        │  ↓ 玩够了想自己创造
        ▼

  💰 转化点：创造种子 = 享客虾会员
├── 用户在享客虾对话中说「我想做颗种子」
├── 享客虾→「自己创造种子需要享客虾会员哦，¥9.9/月，要开通吗？」
├── 用户→开通→创造种子→投池→被领养→开心
└── 用户→不续→继续玩别人种子→下次想创造再续
```

### 付费壁只有一处

| 付费行为 | 价格 | 说明 |
|---------|------|------|
| 自己创造种子 | ¥9.9/月（享客虾会员） | 唯一付费入口 |
| 自定义种子加歌/音乐 | 已包含在会员内 | 无需额外付费 |
| 领养会员专属种子 | 需会员 | 但享受行为本身不额外收费 |

### 没有任何其他收费点

- ❌ 没有按次付费（¥3/首的锚点价已移除）
- ❌ 没有音点/积分
- ❌ 没有多档会员
- ✅ **一个价格（¥9.9/月）覆盖所有创造行为**

### 享客虾自然引导流程图

```
用户说想创造种子
        │
        ▼
享客虾检测会员状态
        │
```
