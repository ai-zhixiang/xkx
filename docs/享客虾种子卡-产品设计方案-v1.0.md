# 🦞 享客虾 × 种子卡 产品设计方案 v1.0

> 让每个人都能种下一颗有生命的创作种子

---

## 一、产品定位

| 维度 | 定位 |
|------|------|
| **一句话** | 享客虾 = 你的AI创作伙伴，种一颗种子，开一支MV |
| **核心场景** | 传几张照片 + 说句话 → 自动生成带音乐的MV |
| **传播引擎** | MV里有你的照片、你的声音——谁看了都有共鸣 |
| **付费模型** | 虾点体系，免费入场，深度使用付费 |
| **技术基底** | 照片→AI识图写诗→Suno配乐→FFmpeg合成MV→RVC换声 |

**核心理念：** 不讲Agent、不讲AI、不讲技术。用户只需要说"我想做个东西给XX看看"，其他交给系统。

---

## 二、种子生命周期

```
                   播种
                    │
    用户传照片 + 说句话
         AI识图 + 写诗 + 配乐
          自动生成MV种子
                    │
                   ▼
                   浇水（次日回访）
                    │
     用户每天回来看——种子在长大
     每次回来 → 数据显示变化（浏览量、播放量）
     虾点浇灌 → RVC换声、高清版
                    │
                   ▼
                   开花
                    │
        MV完整版可分享到朋友圈
        分享即传播 = 朋友看到也想种
                    │
                   ▼
                   结果
                    │
        种子被领养 → 朋友种出TA的版本
        双向连接 → 互相看到
```

### 2.1 播种（用户操作）

用户只需做两件事：

```
① 传照片（1-9张，家人/旅行/孩子/宠物）
② 说句话（按住说话5-30秒:"妈妈我爱你""今天好开心"）

系统自动完成：
├─ AI识图 → 理解照片内容
├─ AI写诗 → 根据照片+语音写一首诗
├─ 风格匹配 → 根据情绪匹配合适的音乐风格
├─ 生成歌曲 → Suno配乐
├─ RVC换声 → 用用户的声音唱（可选）
├─ 合成MV → 照片+歌词+音乐
└─ 种子入库 → 投到种子池
```

### 2.2 浇水（留存机制）— 4天4次惊喜

| 天数 | 用户感知 | 惊喜内容 |
|:----:|----------|----------|
| **Day 1** 🌱 | "种子发芽了！有人刷到了你的种子" | AI自动从你的照片生成一张新风格的海报封面，显示已被3-5人"看过" |
| **Day 2** 🌿 | "长出新叶子！你听——有鸟叫声" | AI根据你的照片情绪生成了一个**15秒配乐预览**，风格跟原版不同（如原版抒情→变成轻快版） |
| **Day 3** 🌷 | "花苞出现了！有人给你的种子点了赞" | 显示"已经有X人说喜欢这颗种子"，解锁**高清缩略图**预览 |
| **Day 4~5** 🌻 | **"开花啦！可以分享给你的朋友了"** | MV完整版可分享，附带"X人种下同款种子"的社交证明 |

### 2.3 开花（传播爆发）

开花后的MV分享出去：
- 照片是用户自己的（亲戚朋友都认识）
- 背景音乐是专门为这组照片写的（独一无二）
- 如果是RVC版，声音也是用户的（熟人猎奇效应）

**这就是妙鸭的"写真即广告"升级版——照片+声音+音乐三重共鸣。**

## 三、虾点体系（算力成本回收）

| 项目 | 消耗 | 说明 |
|------|:----:|------|
| 🆓 每天免费 | 1颗种子 | 基础版MV（标清/30s/无水印版） |
| 🎬 高清完整MV | 10点 | 全高清+60s+可下载 |
| 🎤 RVC换声 | 30点 | 用用户声音唱 |
| 📥 下载MP4 | 5点 | 存到本地 |
| ✍️ 额外写歌 | 50点 | 单独创作一首歌 |
| 💧 浇灌（加速开花） | 3点 | 跳过1天等待 |

### 3.1 定价

| 档位 | 价格 | 虾点 | 日均种子 |
|------|------|:----:|:--------:|
| 🆓 免费 | ¥0 | 0 | 1颗/天 |
| 🎉 特惠月费 | **¥9.9/月** | 3000点 | 3颗/天 |
| 💎 正价月费 | ¥99/月 | 30000点 | 不限 |
| 虾点充值 | 1元=100点 | — | — |

### 3.2 为什么免费用户只能种1颗？

- 1颗刚好够体验完整链路 → 感受到价值
- 想多种/想高清/想换声 → 要虾点 → 付费
- 但免费用户的种子依然能开花、能分享、能传播

**每颗免费种子 = 一次免费病毒传播**，赚了。

## 四、竞争壁垒对照

| 竞品类型 | 我们能做的 | 他们做不到的 |
|----------|-----------|-------------|
| 妙鸭（写真出图） | 照片→MV出视频 | 只会出图，没有音乐+声音 |
| Suno（写歌） | AI写歌+配图+合成 | 只有歌，没有照片+视频 |
| 剪映（视频编辑） | 一句话自动出片 | 要手动操作，门槛高 |
| AI换声App | RVC换声+配MV | 只有声音，没有画面 |
| 星野/猫箱（聊天） | 有产出物（MV） | 聊完啥也没有 |

**享客虾 = 照片 + 音乐 + 声音 + 视频 = 四合一**

## 五、Agent引导设计（首屏爆点）

### 5.1 新用户首屏

```
🦞 享客虾

点「+」传几张照片
说句话
我帮你做成一支MV

[📷 开始创作]  [🎵 看看别人的种子]
```

### 5.2 语音交互

用户按住说话，系统理解意图后自动匹配最佳产出：

| 用户说 | 系统判断 | 产出 |
|--------|----------|------|
| "想给老婆做个生日礼物" | 情感表达→照片MV | 传照片→出MV |
| "今天心情不好" | 情绪宣泄→推荐歌曲 | 推一首歌 |
| "帮我写首歌唱唱我的狗" | 创作→写歌 | 写歌+MV |
| "试试用我的声音" | 功能→RVC引导 | 录语音→RVC换声 |
