## 二、发布前必须完成（P0）

### 1. 🔒 xkx.pangoozn.com 配 HTTPS

**原因：** 微信浏览器限制，HTTP 页面在微信内会被拦截或显示"不安全"

**操作：**
```bash
sudo certbot --nginx -d xkx.pangoozn.com
```

**耗时：** 5 分钟

---

### 2. 📱 微信 IP 白名单

**原因：** Bot 网关调微信公众号 API 时被 40164 拦截
**当前错误：** `invalid ip 139.155.158.18, not in whitelist`
**操作：** 在微信公众平台后台 → 开发 → 基本配置 → IP 白名单，添加 `139.155.158.18`

**耗时：** 2 分钟

---

### 3. 🔑 Hermes API Key 鉴权

**原因：** 当前 Hermes Bridge (:8642) 无鉴权，谁连上都能用
**操作：** 在 weclaw-1/.env 中配置 `HERMES_API_KEY=sk-xxx`，bridge 端启用 key 校验

**耗时：** 10 分钟

---

### 4. 💾 自动备份

**原因：** 数据库无备份，服务器挂了用户数据全丢
**方案：**
- 每日凌晨 3:00 PostgreSQL pg_dump → `/data/backups/`
- 保留最近 7 天
- 通过 cron 执行

**耗时：** 15 分钟

---

### 5. 📊 基础监控

**原因：** 用户 Bot 掉线需要主动发现
**方案：**
- Bot 心跳检测：每 5 分钟发一次 ping，10 分钟无响应告警
- 磁盘 >80% 告警
- 归入 MD-1 现有巡检体系（msgboard 统一告警）

**耗时：** 20 分钟

